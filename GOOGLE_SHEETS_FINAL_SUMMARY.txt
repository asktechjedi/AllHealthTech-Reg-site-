🎯 HOW TO CONNECT GOOGLE SHEETS - 3 KEY IDS
================================================================================

1. SPREADSHEET ID
   Where: URL of your Google Sheet
   Format: https://docs.google.com/spreadsheets/d/{ID}/edit
   Example: 1a2b3c4d5e6f7g8h9i0j
   
2. SERVICE ACCOUNT EMAIL
   Where: JSON credentials file (client_email field)
   Format: name@project-id.iam.gserviceaccount.com
   Example: event-registration-sync@my-project.iam.gserviceaccount.com
   
3. CREDENTIALS FILE
   Where: Downloaded from Google Cloud Console
   Location: backend/credentials/google-sheets-credentials.json
   Format: JSON with private key and authentication info

📋 5-STEP SETUP PROCESS
================================================================================

STEP 1: Create Google Service Account (5 minutes)
   • Go to console.cloud.google.com
   • Create project: "Event Registration Sync"
   • Enable Google Sheets API
   • Create service account
   • Download JSON credentials

STEP 2: Create Google Spreadsheet (5 minutes)
   • Go to sheets.google.com
   • Create spreadsheet: "Event Registrations"
   • Share with service account email
   • Copy spreadsheet ID from URL

STEP 3: Place Credentials File (2 minutes)
   • mkdir backend/credentials
   • cp credentials.json backend/credentials/google-sheets-credentials.json
   • Add credentials/ to .gitignore

STEP 4: Configure .env (3 minutes)
   • GOOGLE_SHEETS_ID=your_id_here
   • GOOGLE_SHEETS_CREDENTIALS_PATH=./credentials/google-sheets-credentials.json
   • GOOGLE_SHEETS_SHEET_NAME=Registrations

STEP 5: Test & Verify (5 minutes)
   • npm run db:migrate
   • npm run dev
   • Create test registration
   • Check Google Sheets for new row

TOTAL TIME: 20-30 minutes

================================================================================
🔄 AUTOMATIC RETRY LOGIC
================================================================================

If sync fails (network issue, rate limit, etc.):

Attempt 1 → Fails
  ↓ Wait 1 second
Attempt 2 → Fails
  ↓ Wait 2 seconds
Attempt 3 → Fails
  ↓ Wait 4 seconds
Attempt 4 → Fails
  ↓ Dead Letter Queue (manual review)

Transient errors (will retry):
   • Network timeouts
   • DNS failures
   • Rate limits (429)
   • Server errors (5xx)

Permanent errors (won't retry):
   • Authentication failed (401)
   • Permission denied (403)
   • Not found (404)
   • Invalid credentials

================================================================================
❌ TROUBLESHOOTING
================================================================================

Issue: "Failed to load credentials"
   → Check file exists at backend/credentials/google-sheets-credentials.json

Issue: "Authentication failed (401)"
   → Verify spreadsheet is shared with service account email

Issue: "Sheet not found (404)"
   → Check sheet name matches exactly (case-sensitive)

Issue: Registrations not syncing
   → Verify GOOGLE_SHEETS_ID is set in .env

Issue: Rate limit errors (429)
   → System retries automatically; increase delay if needed
