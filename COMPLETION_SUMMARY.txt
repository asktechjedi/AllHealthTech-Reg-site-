================================================================================
                         HOW IT WORKS
================================================================================

REGISTRATION FLOW:
  1. User fills registration form
  2. Backend validates data
  3. Database saves registration
  4. Confirmation email sent (async)
  5. Google Sheets sync triggered (async)
  6. Success page with ticket ID
  7. Option to register another account

GOOGLE SHEETS SYNC:
  1. Registration created
  2. Sync triggered automatically
  3. Credentials loaded
  4. Google API authenticated
  5. Data mapped to sheet row
  6. Row appended to Google Sheet
  7. Success logged

ERROR HANDLING:
  Transient Errors (network, rate limit, 5xx):
    → Queued for retry
    → Exponential backoff (1s, 2s, 4s)
    → Automatic retry
    → Success or permanent error

  Permanent Errors (auth, 404):
    → Moved to dead letter queue
    → Manual review required
